# Mesh

Mesh是用于将几何体（Geometry）和材质（Material）结合起来创建 3D 对象的类。Mesh 是 THREE.js 的核心类之一，它可以通过将几何体与材质进行组合来创建具有颜色、纹理、透明度等特性的 3D 模型。 Mesh对象不仅仅是一个3D模型，它还可以添加动画、事件等功能，使其更具交互性和动态性。

# 示例

var geometry = new THREE.BoxGeometry(1, 1, 1)
var material = new THREE.MeshBasicMaterial({ color: 0xff0000 })
var mesh = new THREE.Mesh(geometry, material)
State.scene.add(mesh)
