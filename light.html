# 灯光(Light)

three.js 库中提供了多种类型的灯光，用于模拟不同类型的光源。常见的灯光类型包括：

THREE.PointLight：点光源，类似于电灯泡，会向四周发射光线，能够产生阴影。

THREE.DirectionalLight：方向光源，类似于太阳光，光线是平行的，不会产生阴影。

THREE.SpotLight：聚光灯，类似于手电筒，可以聚焦光线，产生锥形的光照范围，能够产生阴影。

THREE.HemisphereLight：半球光源，用于模拟天空的光线，可以产生柔和的环境光效果，不会产生阴影。

THREE.AmbientLight：环境光源，用于增加场景的整体亮度，不会产生阴影。

# 环境灯(AmbientLight)

THREE.AmbientLight是three.js库中的一个类，它表示一种环境光源，用于在场景中模拟自然光线照射的效果。环境光源是一种全局光源，不会产生阴影，而是均匀地照亮整个场景。它通常用于增加场景的整体亮度，使得场景中的物体看起来更加自然。

在创建THREE.AmbientLight实例时，可以指定光源的颜色和强度。颜色可以是十六进制数值或THREE.Color对象，强度则是一个标量值，默认为1。在将THREE.AmbientLight实例添加到场景中后，场景中的所有物体都将受到该光源的影响。

# 创建环境灯

let light = new THREE.AmbientLight(0xffffff, 1)
State.scene.add(light) // 将灯光添加进场景内

# 平行光(DirectionalLight)

THREE.DirectionalLight是three.js库中的一种灯光类型，用于模拟方向光源，例如太阳光。方向光源是一种平行的光源，光线方向不受位置影响，因此不需要指定光源的位置。与点光源不同，方向光源不会向四周发射光线，因此不会产生阴影。

在创建THREE.DirectionalLight实例时，需要指定光源的颜色和强度。与其他灯光类型不同，方向光源不需要指定位置，而是需要指定光线的方向，可以通过设置光线方向向量的方式来实现。可以通过设置target属性，将光线的方向对准场景中的一个对象，以便更好地控制光线的方向。

# 创建平行光

  let light1 = new THREE.DirectionalLight(0xff0000, 10)
  State.scene.add(light1) // 将灯光添加进场景内
