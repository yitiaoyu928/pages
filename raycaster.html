# 射线拾取(raycaster)

Raycaster是 three.js 库中的一个类，用于进行射线拾取操作，即检测射线与场景中的物体是否相交，并返回相交的物体信息。射线拾取是一种常用的交互方式，可以用于实现鼠标交互、物体拖拽、碰撞检测等功能。

Raycaster 需要传入两个参数，一个是射线的起点，另一个是射线的方向。可以通过设置相应的属性来控制射线的起点和方向，例如设置 raycaster.setFromCamera()方法的参数，可以将射线的起点设置为相机的位置，并将射线的方向设置为相机指向鼠标点击位置的方向。

Raycaster 的 intersectObjects()方法可以用于检测射线和场景中的物体是否相交，并返回相交的物体信息。该方法需要传入一个参数，即需要检测的物体数组。如果射线与某个物体相交，则该方法会返回一个包含相交物体信息的数组，包括相交点的坐标、相交面的法向量、相交的物体等。

# 创建

const raycaster = new THREE.Raycaster()
const mouse = new THREE.Vector2()
// 屏幕坐标转空间坐标
function onMouseMove(event) {
  mouse.x = (event.clientX / window.innerWidth) * 2 - 1
  mouse.y = -(event.clientY / window.innerHeight) * 2 + 1
}

function pickObjects() {
  // 设置射线的起点和方向
  raycaster.setFromCamera(mouse, camera)
  // 检测射线和场景中的物体是否相交。如果相交，则将相交物体的名称输出到控制台中

  if (intersects.length > 0) {
    const intersect = intersects[0]
    console.log(intersect.object.name)
  }
}

window.addEventListener('click', pickObjects, false)
