# 几何形状(geometry)

THREE.js 是一种用于在 Web 浏览器中创建和渲染 3D 图形的 JavaScript 库。它包含了许多用于处理 3D 模型的工具和功能，包括多种预先构建的几何体，用于创建网格。

BoxGeometry：创建一个带有可定制宽度、高度和深度的矩形盒子。

SphereGeometry：创建一个带有可定制半径和分段数的球体。

CylinderGeometry：创建一个带有可定制半径、高度和分段数的圆柱体。

PlaneGeometry：创建一个平面，带有可定制宽度和高度。

TorusGeometry：创建一个带有可定制半径、管道直径、径向分段数和管道分段数的圆环体。

TubeGeometry：创建一个带有可定制半径、管道直径和分段数的管道。

ExtrudeGeometry：通过沿路径挤压 2D 形状来创建 3D 形状。
