# 渲染器(render)

WebGL 是一种用于 Web 的低级别图形 API。WebGLRenderer 是 three.js 的核心组件之一，负责创建 WebGL 上下文，设置渲染管道，并将场景绘制到屏幕上

# 创建

State.render = new THREE.WebGLRenderer({ canvas: canvasRef.value })
State.render.setSize(window.innerWidth, window.innerHeight)
State.render.setPixelRatio(window.devicePixelRatio) // 设置屏幕像素比,当页面缩放后效果会明显一些
State.render.render(State.scene, State.camera) // 渲染场景和相机到canvas上
