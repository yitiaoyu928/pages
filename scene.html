# 场景介绍

场景（Scene）是一个容器，用于存放所有的三维对象、光源和摄像机。它是构建 Three.js 应用的基础，并且是所有 Three.js 应用的必要组成部分。

# 创建

State.scene = new THREE.Scene()
1

# 背景色

State.scene.backgroundColor = new THREE.Color(0x000000) // THREE.Color 接收十六进制代码和css颜色字符串
